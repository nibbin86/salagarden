Easy UTM tracking with contact form 7 submits UTM leads & referrer parameters in your email with single click installation.

== Description ==


A simple plugin to give you the power to track UTMs on your contact form 7 using just one click enabling. 
You receive the UTM parameters & referrer in every lead email right into your inbox. 

This plugin saves any UTM values coming in url and let the users browse the website normally. 
You dont have to worry about passing the values when users are switching from page to page. All the values are stored and automatically retrived when a user submits any contact form 7. 

No additional setups are required. One click installation works on all the cf7 forms automatically. 

**Following are the utm tags you will track**

* utm_source 
* utm_medium 
* utm_term 
* utm_content 
* utm_campaign 
* Landing page URL
* Referrer (traffic source) 


== Installation ==

Installation using wordpress: 

1. Login into admin panel. 
2. Go to plugins. 
3. Click on add new.  
4. In search field search for 'Easy UTM tracking with contact form 7'. 
5. Cick install.
6. Enable the plugin after you see successfull installation page. 

Installation by downloading the plugin:

1. Download the plugin from wordpress.
2. Extract the plugin files into `/wp-content/plugins/` directory using ftp or by uploading directly the zip file from plugins page.
3. Visit plugins page in your wordpress admin panel.
4. Activate the plugin through the 'Plugins' screen in WordPress
5. Happy UTM tracking
